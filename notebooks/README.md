# Notebook Catalog

These notebooks are included as narrative analysis records. They are not all required for a clean re-run, but they document how the results were inspected and interpreted.

- `cs514_data_collection_walkthrough.ipynb` describes BGG API access and the collection workflow.
- `cs514_bgg_eda.ipynb` summarizes early exploratory data analysis.
- `cs514_parameter_refinement_walkthrough.ipynb` explains the alpha and gamma refinement process.
- `cs514_community_interpretation.ipynb` inspects community labels, representative games, and enrichment signals.
- `cs514_structural_network_analyses.ipynb` covers centrality, topology, metadata alignment, and community-level structure.
- `cs514_user_archetype_analysis.ipynb` analyzes user taste profiles and archetype families.
- `cs514_nmf_test_a.ipynb` interprets the NMF recovery test for the 11 taste dimensions.
- `cs514_nmf_test_b.ipynb` interprets the full-matrix NMF k-sweep.
- `cs514_network_findings.ipynb` gathers selected final network findings.
